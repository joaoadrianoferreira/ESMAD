# RESTful API Boilerplate

RESTful API example using NodeJS with express and mongoose. 

## Instructions

    ## Clone Repository ##
    git clone https://github.com/joaoadrianoferreira/ESMAD.git
    cd ESMAD
    git checkout graphql
    
    ## Install Node Modules ##
    npm install 
    
    ## Start Application ##
    npm start
